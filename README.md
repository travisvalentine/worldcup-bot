## Endpoints:

###### Team names:

```
/teams
```

###### Today's Matches:

```
/matches
```

###### Tomorrow's Matches:

```
/matches/tomorrow
```

###### Today's Scores:

```
/scores
```

###### Team's Past/Current Scores:

```
/scores/:country_short_name
```

###### Groups Standings:

```
/groups/:group
```
